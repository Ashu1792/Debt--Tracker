# 💰 Debt Tracker - Flask Web Application

A simple and professional **Debt Management Web Application** built using **Flask, SQLite, Bootstrap, and Python**.
This platform helps users **track money lent to people, monitor payments received, and manage remaining balances** through a clean dashboard interface.

# 🚀 Live Demo

https://debt-tracker-0dlk.onrender.com


# 📌 Features

✅ Add new debt records
✅ Track total amount lent
✅ Track total amount received
✅ Automatically calculate remaining balance
✅ Record partial payments
✅ Update payment dates
✅ Delete debt records
✅ Professional dashboard UI
✅ Responsive design using Bootstrap
✅ SQLite database integration

# ⚙️ Installation & Setup

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```
git clone https://github.com/YOUR_USERNAME/Debt--Tracker.git
```

### 2️⃣ Navigate to project folder

```
cd Debt--Tracker
```

### 3️⃣ Create virtual environment

```
python -m venv venv
```

Activate environment

**Windows**

```
venv\Scripts\activate
```

**Mac/Linux**

```
source venv/bin/activate
```

### 4️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 5️⃣ Run the application

```
python app.py
```

### 6️⃣ Open browser

```
http://127.0.0.1:5000
```

# 📊 How It Works

1. User adds a debt record with:

   * Debtor name
   * Amount
   * Lending date

2. The system stores data in **SQLite database**.

3. Users can update payments when money is received.

4. Dashboard automatically calculates:

* Total Lent
* Total Received
* Remaining Balance
